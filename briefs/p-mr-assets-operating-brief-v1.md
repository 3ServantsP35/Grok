## 1. Purpose

The **MR Assets Operating Brief** is a decision-first artifact for evaluating the **MR Assets** theme as a cross-asset regime and capital-allocation map.

Its purpose is to answer two main questions:

1. **Overall MR Assets attractiveness**  
   How attractive is the MR Assets opportunity set relative to:
   - other themes
   - the benchmark / reserve alternative, **STRC**

2. **Optimal internal composition**  
   If capital should move away from passive reserve posture, where across the **MR Assets** cross-asset composite is it best treated?

This should not be treated as a generic macro note.

It is an **Operating Brief** intended to produce decision-ready clarity about where fresh dollars are best treated across macro / regime-sensitive assets.

---

## 2. Core question

> Is the MR Assets universe attractive enough, relative to other themes and STRC, to deserve capital now, and if so, what is the best current cross-asset composition inside it?

That breaks into two decisions:
- **theme selection**
- **cross-asset composition**

---

## 3. What MR Assets is

MR Assets is not a single thesis basket like Visser.

It is better understood as a **macro / regime opportunity complex** made up of cross-asset sleeves that help answer:
- where is macro-sensitive opportunity strongest?
- what kind of assets are currently treating new dollars well?
- which regime expressions are improving, neutral, or deteriorating?

Based on the live theme scope, MR Assets currently includes sleeves such as:
- **rates** (`US02Y`, `US10Y`, `TLT`)
- **credit** (`HYG`, `LQD`, `HYG_LQD`)
- **equities / leadership proxies** (`QQQ`, `VT`, `XLE`, `XLK`, `XLF`, `XLP`, `XLY`, `IWM_SPY`, `QQQ_SPY`)
- **dollar / FX / vol** (`DXY`, `USDJPY`, `VIX`)
- **commodities / metals / cyclicals** (`DBC`, `HG1`, `SI1`, `CL1`, `NG1`, `FCX`, `SLV`, `GLD_DXY`, `COPPER_GOLD`)
- **crypto / liquidity / breadth proxies** (`BTCUSD`, `ETHUSD`, `TOTAL`, `TOTAL2`, `BTC_D`, `ETH_D`, `BTC_QQQ`, `STABLE_C_D`)

That means MR Assets should not primarily ask:
- is one narrative theme compelling?

It should ask:
- which macro-sensitive asset families are currently the best homes for capital?
- which are warning, congested, or unattractive?
- what cross-asset mix best expresses the current opportunity set?

## 4. Position in the architecture

The MR Assets Operating Brief should consume:
- **Layer 0** macro backdrop
- **Layer 0.5** Howell phase / deployment posture
- **Layer 0.75** APE thematic intelligence where relevant, but usually with lighter weight than in Visser
- **Layer 1** structural trust / gate state
- **MR Assets theme-state data** from the warehouse

It should be:
- architecture-aware
- cross-asset
- comparative
- benchmark-aware

It should remain:
- independent from MSTR-specific thesis work
- independent from Visser-style thematic underwriting
- focused on where macro/regime opportunity is strongest

## 5. What the brief must deliver

Every MR Assets Operating Brief should produce two outputs.

### 5.1 Output A — MR Assets attractiveness versus alternatives

The brief should judge whether the MR Assets opportunity set is:
- more attractive than competing themes
- less attractive than competing themes
- attractive enough to beat **STRC** as benchmark / reserve alternative
- not attractive enough to justify leaving reserve posture aggressively

This output should end in a usable judgment such as:
- **High-conviction add**
- **Add**
- **Selective add**
- **Hold / maintain**
- **Selective profit-take**
- **Harvest / reduce**
- **Defer / prefer STRC**

### 5.2 Output B — Optimal cross-asset composition

The brief should also identify the best internal **cross-asset composite**.

This means it should answer:
- which sleeves are the best current homes for capital?
- which sleeves are structurally sound but tactically less attractive?
- which sleeves should be held, not expanded?
- which sleeves are in trim / avoid territory?
- where should capital rotate if regime conditions shift?

This matters because MR Assets is fundamentally a **relative opportunity map** across asset classes.

## 6. Draft operating logic

### 6.1 Theme-level judgment

At the theme level, MR Assets should judge whether the macro / regime opportunity set deserves capital relative to:
- peer themes
- **STRC** as benchmark / reserve alternative

Questions to answer:
- are macro / regime assets currently treating fresh dollars well enough to leave reserve posture?
- is the opportunity broad enough to justify real deployment?
- or is the correct answer that STRC remains the better home for capital?

### 6.2 Composite-level judgment

At the composite level, the brief should rank the internal sleeves by attractiveness.

Questions to answer:
- where does the next dollar belong?
- which sleeves are improving?
- which are mature or congested?
- which are deteriorating?
- what rotation path is implied if conditions evolve?

## 7. First draft of the MR Assets attractiveness engine

A future MR Assets engine should probably measure at least these dimensions:

### A. Relative attractiveness versus alternatives
- does MR Assets deserve capital more than other themes?
- does it deserve capital more than STRC?

### B. Cross-asset breadth
- is opportunity broadening across multiple sleeves?
- or narrowing into only one or two isolated pockets?

### C. Regime alignment
- do the strongest sleeves align with the current macro / Howell posture?
- or do they fight the broader architecture?

### D. Leadership quality
- which sleeves are actually carrying the opportunity set?
- are those leaders durable, or increasingly fragile?

### E. LOI / fresh-dollar friendliness
- where available, is **AB3 LOI** suggesting price is attractive for fresh dollars?
- where LOI is missing, does the brief need to downgrade confidence?

### F. Congestion / digestion / stretch
- are the leading sleeves still attractive?
- or becoming crowded enough that STRC or a different sleeve is preferable?

### G. Defensive alternative quality
- how compelling is simply remaining in **STRC** relative to pressing macro/regime exposures?

### H. Rotation clarity
- is the next move obvious?
- or is the opportunity set too mixed to justify aggressive reallocation?

## 9. Specific design difference versus Visser

Visser is mainly a **thematic thesis and composite-underwriting brief**.

MR Assets should be different.

MR Assets should be mainly a:
- **cross-asset attractiveness brief**
- **macro / regime deployment map**
- **relative-opportunity allocator**

The main question is less:
- what thesis family is most compelling?

and more:
- what type of assets are currently the best homes for incremental capital?

That distinction should stay explicit.

---

## 10. Early view on AB3 LOI usage

AB3 LOI may become especially useful in MR Assets because this brief is closely tied to the question:
- where are **fresh dollars** best treated?

So the likely rule should be:
- use LOI as a major timing / entry-attractiveness input where it is populated and trustworthy
- downgrade confidence where LOI coverage is sparse or null
- do not let missing LOI values force false precision in sleeve ranking

This is likely more central to MR Assets than to some other briefs, because incremental-dollar friendliness is very close to the brief’s core purpose.
